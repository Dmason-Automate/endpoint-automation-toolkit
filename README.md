# Endpoint Automation Toolkit

This repository contains PowerShell scripts and documentation used to automate and improve endpoint management processes in Windows environments.

## Focus Areas
- Device provisioning and imaging
- Automation of repetitive IT tasks
- Endpoint management (Intune)
- Process improvement and efficiency

## Tools & Technologies
- PowerShell
- Microsoft Intune
- Windows Deployment

## Purpose
The goal of this project is to reduce manual effort, improve deployment consistency, and streamline IT support workflows.

## Notes
All examples are generalized and do not contain company-specific or sensitive information.
